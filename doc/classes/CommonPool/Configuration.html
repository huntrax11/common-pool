data_source  [RW]  The data source object to be extended, code>create_object</code> method need to be overridden
idle_check_interval  [RW]  Eviction thread interval in seconds. Default: 0. If set to 0 or less, the eviction thread will not run.
idle_check_no_per_run  [RW]  Number of objects to check per eviction thread run. Default: 3. If set to 0 or less, all idle objects will be checked.
logger  [RW]  Assign a logger, to log pool debug/info messages, used this for debugging purpose.
max_active  [RW]  Maximum number of active objects in the pool. Default: 8.
max_idle  [RW]  Maximum number of idle objects in the pool. Default: 8.
max_idle_time  [RW]  Maximum idle time in seconds, before an object can be evicted. Default: 1800.
min_idle  [RW]  Minimum number of idle objects in the pool. Default: 0.
timeout  [RW]  Request timeout when creating object for the pool, timeout in seconds. Default: 60. If set to 0 or less, there will be no timeout.
validation_timeout  [RW]  Request timeout when validation idle object in the pool, timeout in seconds. Default: 30. If set to 0 or less, there will be no timeout.
